::: tip
JavaScript 3种引入方式
:::
<!--more-->
##  行内

```html
<a href="javascript:alert('hello world')">点击</a>
```

```html
<div onclick="window.alert('Hello world!')"></div>
```



##  内联

```html
<script type="text/javascript">
    alert('hello world');
</script>
```

##  外链式

```html
<script src="./01.js" type="text/javascript"></script>
```
