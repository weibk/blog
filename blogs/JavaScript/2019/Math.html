JavaScript 中的数学对象(Math)

# min-max的随机数

Math.floor(Math.random()*(max-min+1))+min
方法 描述
abs(x) 返回 x 的绝对值。
acos(x) 返回 x 的反余弦值。
asin(x) 返回 x 的反正弦值。
ceil(x) 对数进行上舍入。
cos(x) 返回数的余弦。
exp(x) 返回 Ex 的指数。
floor(x) 对 x 进行下舍入。
log(x) 返回数的自然对数（底为e）。
max(x1,x2,x3,x4....) 返回 x,y,z,...,n 中的最高值。
main(x1,x2,x3,x4...) 返回 x,y,z,...,n中的最低值。
pow(x,y) 返回 x 的 y 次幂。
random 返回 0 ~ 1 之间的随机数。
round(x) 四舍五入。
sin(x) 返回数的正弦。
tan(x) 返回数的平方根。
sqrt(x) 返回角的正切。
